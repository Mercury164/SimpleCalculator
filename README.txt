Simple Calculator (Android Studio, Java)

How to open:
1. Extract the archive.
2. Open the project folder in Android Studio.
3. Wait for Gradle Sync to finish.
4. Run on an emulator or Android device.

Features:
- Basic operations: +, -, ×, ÷
- Decimal numbers
- Clear and delete buttons
- Division by zero protection

Notes:
- The project uses Java + ViewBinding.
- Android Studio may suggest updating the Android Gradle Plugin or Gradle wrapper depending on your local version.
